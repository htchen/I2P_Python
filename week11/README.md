## Sample Practice Problems

### Problem 1: Sorting
Given a list of places with ratings and distances, sort by rating (descending), then by distance (ascending) for ties.

### Problem 2: Filtering
Filter a list of restaurants to only include those with:
- Rating >= 4.0
- Walking time <= 10 minutes
- Category == "pizza"

### Problem 3: Optimization
Given a 4x4 distance matrix, find the shortest route visiting all locations.

### Problem 4: Combined
Write a function that takes a list of places and returns the top 3 highest-rated places within 15 minutes walking distance, in optimal visiting order.
