# 第十一週 期中考筆試題目

**考試形式：** 筆試（5%）
**範圍：** 程式設計基本概念（與 Python 語法無關）

---

## 第 1 題：可變 vs. 不可變資料型態

請回答下列問題：

(a) 請舉出 Python 中**兩種「不可變（immutable）」**的資料型態或資料結構。

(b) 請舉出 Python 中**兩種「可變（mutable）」**的資料型態或資料結構。

(c) 請說明「可變」與「不可變」資料在程式設計上有什麼差異？並舉一個因為使用可變資料而可能產生 bug 的情境。

---

## 第 2 題：純函式（Pure Function）

什麼是「純函式（Pure Function）」？為什麼 `sorted(my_list)` 比 `my_list.sort()` 在程式設計上更接近「純」的概念？這種寫法在維護程式碼時有什麼好處？

---

## 第 3 題：錯誤處理的設計觀念

程式中遇到「網路斷線」、「API 回傳 404」、「JSON 格式錯誤」等情況時，為什麼通常使用 `try / except` 而不是用 `if` 來事先判斷？請比較這兩種風格的差異與各自的適用時機。

---

## 第 4 題：函數式三劍客（map / filter / reduce）

請用一句話分別說明 `map`、`filter`、`reduce` 三者的用途與差異。並說明為什麼這種寫法被稱為「宣告式（declarative）」，它與傳統 `for` 迴圈在「描述程式」的方式上有什麼不同？

---

## 第 5 題：進位制轉換

請完成下列轉換（需寫出計算過程）：

(a) 將十進位 `156` 轉換為**二進位**與**十六進位**。

(b) 將二進位 `1011 0110` 轉換為**十進位**與**十六進位**。

(c) 將十六進位 `0x2F` 轉換為**十進位**與**二進位**。

(d) 為什麼程式設計中常用「十六進位」來表示二進位數值？（請從「易讀性」的角度說明。）
